Demo Class Preparation

Schools / institutes demo me aapka delivery, confidence aur classroom control check karte hain.

10–15 min Ideal demo duration
Clarity > Depth Simple examples win
Engagement Question every 60–90s
Best Demo Class Structure (Simple Template)
1
Hook (30–60s)
Real-life example / quick question: “Why do we need this topic?”
2
Concept (3–5 min)
Define clearly, 1–2 key rules, board/notes neat.
3
Example (4–6 min)
Easy → medium. Ask students “next step kya hoga?”
4
Practice (2–3 min)
2 quick questions. Ask 2–3 students (or imagine responses).
5
Recap (30–60s)
3 bullet summary + homework/next topic hint.

Demo Checklist (Must Do)
Common Mistakes (Avoid)
  • Only lecture, no student involvement
  • Too advanced example first
  • Messy board / unstructured teaching
  • Rushing due to time mismanagement
  • Arguing or getting angry (even if students “disturb”)
One Line to Remember
Institutes often judge communication + clarity + control more than deep subject content. Keep it structured, engaging, and calm.
